¿Que es HTML5?

Esto fue posteado el 10/07/2018 por UMG-Japala

HTML5 es la última versión de HTML (HyperText Markup Language), el lenguaje de marcado estándar para documentos web. HTML5 es una respuesta a la observación de que la versión anterior de HTML (HTML 4.01) no se actualizó desde 1999 y no puede manejar las demandas de las aplicaciones web contemporáneas. Es un intento de definir una sola versión de HTML que pueda ser escrita o interpretada de la misma manera en todos los navegadores. HTML5 se basa en tres pilares principales: un nuevo conjunto de elementos semánticos, la capacidad de incluir contenido multimedia sin complementos y una API de JavaScript mejorada.

¿Que es CSS3?

Esto fue posteado el 10/07/2018 por UMG-Japala

CSS3 es la última evolución del lenguaje de hojas de estilo en cascada y pretende ampliar la especificación CSS2.1. Introduce muchas novedades y nuevas características como selectores avanzados, efectos de fondo, bordes redondeados o sombras, mejoras en la navegación y la interactividad, entre otros.

Con CSS3 se pueden crear efectos visuales y estilos avanzados para mejorar la apariencia de las páginas web. Además, permite separar el contenido de la presentación y la estructura de los documentos web.
